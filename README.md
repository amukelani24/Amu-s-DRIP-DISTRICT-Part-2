Project Overview
Amu's DRIP DISTRICT is a modern streetwear fashion website created to showcase urban fashion collections, fashion branding, and online customer interaction.
The website includes multiple pages:
Home Page
About Page
Gallery Page
Contact Page
Services Page
The purpose of the website is to present a professional fashion brand using HTML and CSS while applying responsive web design principles.
